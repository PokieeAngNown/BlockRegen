**插件版本: 1.0.7**  
**支持服务器核心: Spigot/Paper**  
**支持游戏版本: 1.20.1 [其余版本还未测试]**

***遇到影响到插件本身功能的问题请在github发布issue或者在kook群组服插件频道提出***

---
插件内容:

- 自定义区域内的方块重新生成
- 自定义方块按照哪种列表生成
- 自定义语言文件
